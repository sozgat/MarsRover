# Mars Rover

## Installation Guide
Mars Rover Application can run on Docker easily. Please follow below commands on your terminal.

```bash
git clone https://github.com/sozgat/MarsRover.git
cd MarsRover
docker build -t marsrover .
docker run -it marsrover
```
